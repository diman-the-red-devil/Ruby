## 2. Комментарии

Шаги:

1. Добавить код в файл **main.rb**

* Однострочный комментарий

```ruby
# Однострочный комментарий
# puts выводит строку
puts "This Is A String"
```

* Многострочный комментарий

```ruby
=begin
Многострочный комментарий
puts выводит строку
=end
puts "This Is A String"
```

2. Полный код программы

3. Запустить программу

Результат

